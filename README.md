# 学习记录：多语言（learn-multiple-langs）

这个仓库用于记录我（SZCCD）的编程学习历程、笔记与练手项目。目标是逐步积累：
- 语言知识（Python、JavaScript 等）
- 小项目与练习题
- 学习笔记与问题总结

目录结构（示例）
- python/        — Python 练习与笔记
- js/            — JavaScript 练习与笔记
- projects/      — 可运行的小项目
- notes/         — 读书笔记、学习日志

使用说明
1. 每次开始新主题或练习，创建分支 `topic/<language>-<short-topic>` 并在完成后合并到主分支。
2. 保持 README 与子目录 README 更新，方便回顾。
3. 推荐使用 `issues` 跟踪练习目标与 bug。

每天都进步！
